## 📖 Інструкція для користувача

### Структура графіків

| Графік | Що відображає |
|---|---|
| **Верхній** | Чиста гармоніка (синя) + зашумлена (помаранчева) |
| **Нижній** | Зашумлений вхід (помаранчева прозора) + відфільтрована (зелена) + чиста-референс (синя пунктирна) |

### Слайдери

| Слайдер | Ряд | Що змінює |
|---|---|---|
| **Амплітуда (A)** | 1 | Висота піків гармоніки `[0.1 … 5.0]` |
| **Частота (f), Гц** | 1 | Кількість циклів за секунду `[0.1 … 10.0 Гц]` |
| **Фаза (φ), рад** | 1 | Зсув гармоніки по горизонталі `[0 … 2π рад]` |
| **Серед. шуму (μ)** | 2 | Зміщення шуму по вертикалі `[-1.0 … 1.0]` |
| **Дисперсія шуму (σ²)** | 2 | Інтенсивність (розкид) шуму `[0.01 … 1.0]` |
| **Зріз фільтра, Гц** | 3 | Гранична частота low-pass фільтра `[0.5 … 20.0 Гц]` |

> ⚠️ **Важлива логіка:**
> - Зміна **рядка 1** (гармоніка) → шум **не регенерується**
> - Зміна **рядка 2** (шум) → шум **регенерується**, параметри гармоніки зберігаються

### Чекбокси

- **«Показати шум»** - вмикає/вимикає зашумлену лінію на верхньому графіку. Якщо вимкнено - видно лише чисту гармоніку.
- **«Показати фільтр»** - вмикає/вимикає відфільтровану лінію на нижньому графіку.

### Кнопка Reset

Відновлює **всі** параметри до початкових значень і регенерує шум.

### Рекомендовані експерименти

1. Збільшіть **дисперсію шуму** до `0.8` → поспостерігайте, як фільтр очищує сигнал.
2. Зменшіть **зріз фільтра** нижче за частоту сигналу → фільтр спотворить гармоніку.
3. Збільшіть **частоту** до `5 Гц`, зріз фільтра залиште на `3 Гц` → гармоніка буде відфільтрована разом з шумом.
