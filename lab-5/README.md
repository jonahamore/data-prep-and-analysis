# Лабораторна робота №5 - VHI Streamlit Dashboard

## Опис завдання

Веб-додаток для інтерактивного аналізу та візуалізації індексів вегетаційного здоров'я (VHI, TCI, VCI) по регіонах України на основі даних NOAA (1981–2024).

---

## Основні етапи роботи

1. **Завантаження даних** - автоматичне завантаження CSV-файлів з NOAA для кожної з 27 областей України з кешуванням локально.
2. **Зчитування та очищення** - об'єднання файлів у єдиний DataFrame, заміна некоректних значень `-1` на `NaN`, видалення порожніх рядків, додавання стовпців з індексом та назвою області.
3. **Веб-додаток** - інтерактивний інтерфейс зі слайдерами, dropdown-списками, чекбоксами та кнопкою скидання фільтрів.
4. **Фільтрація** - за областю, інтервалом тижнів та інтервалом років.
5. **Сортування** - за зростанням або спаданням обраного показника.
6. **Відображення** - три вкладки: таблиця, графік часового ряду, порівняння областей.

---

## Основні процедури

- **`load_all_data()`** - завантажує CSV-файли з NOAA для всіх областей України та кешує їх локально у папку `vhi_data/`. При повторному запуску дані зчитуються з кешу без звернення до мережі.

- **`_parse_vhi_csv(content, region_id)`** - парсить сирий текст відповіді NOAA у pandas DataFrame. Видаляє HTML-рядки, замінює некоректні значення на `NaN`, додає стовпці `region_id` та `region`.
  - `content` - текст відповіді сервера
  - `region_id` - числовий індекс області (1–27)

- **`_init_state(year_min, year_max)`** - ініціалізує `st.session_state` початковими значеннями всіх фільтрів при першому завантаженні сторінки.

- **`_reset(year_min, year_max)`** - скидає всі інтерактивні елементи до початкового стану:
  - обраний показник повертається до `VHI`
  - обрана область повертається до `Київська`
  - інтервал тижнів повертається до `(1, 52)`
  - інтервал років повертається до повного діапазону
  - обидва чекбокси сортування скидаються у `False`

- **`_chart_timeseries(df, metric, region, week_range, year_range)`** - будує лінійний графік значень обраного показника для вибраної області за вказаним діапазоном років та тижнів.

- **`_chart_comparison(all_df, filtered_df, metric, region, week_range, year_range)`** - будує графік порівняння середніх значень обраного показника по всіх областях. Обрана область виділяється червоним кольором, інші - сірим.

## Функціонал веб-додатку

| Елемент | Опис |
|---|---|
| Dropdown - показник | Вибір між VCI, TCI або VHI |
| Dropdown - область | Вибір однієї з 27 областей України |
| Slider - тижні | Фільтрація за інтервалом тижнів (1–52) |
| Slider - роки | Фільтрація за інтервалом років (1981–2024) |
| Checkbox ↑ зростання | Сортування таблиці за зростанням обраного показника |
| Checkbox ↓ спадання | Сортування таблиці за спаданням обраного показника |
| Кнопка скидання | Повернення всіх фільтрів до початкових значень |

> Якщо увімкнені **обидва** чекбокси одночасно - відображається попередження, застосовується сортування за зростанням.

## Вкладки результатів

### Таблиця

Відфільтровані дані у вигляді таблиці. Містить стовпці:

| Стовпець | Опис |
|---|---|
| `region` | Назва області |
| `year` | Рік |
| `week` | Номер тижня |
| `vci` | Vegetation Condition Index |
| `tci` | Temperature Condition Index |
| `vhi` | Vegetation Health Index |

### Часовий ряд

Лінійний графік значень обраного показника (VCI / TCI / VHI) для вибраної області. Будується за:
- вибраною областю
- вибраним показником
- вибраним інтервалом років
- вибраним інтервалом тижнів

### 🗺️ Порівняння областей

Графік порівняння середніх значень обраного показника між усіма областями за вказаний часовий інтервал. Особливості:
- враховується вибраний інтервал років та тижнів
- обрана область виділяється **червоним** кольором
- інші області відображаються **сірим** кольором

## Встановлення та запуск

```bash
# 1. Перейти до папки лабораторної
cd lab_05

# 2. Створити та активувати віртуальне середовище
python -m venv .venv

# Windows:
.venv\Scripts\activate
# macOS / Linux:
source .venv/bin/activate

# 3. Встановити залежності
pip install -r requirements.txt

# 4. Запустити додаток
streamlit run app.py
```

Після запуску браузер автоматично відкриє `http://localhost:8501`.
