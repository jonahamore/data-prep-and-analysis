# Шоста лабораторна робота

## Опис завдання

Потрібно було створити C++ програму з використанням OpenCV для роботи з відеопотоком з камери, обробки зображення в реальному часі та взаємодії з користувачем через клавіатуру, мишу і слайдер.

Основні етапи роботи:

* Створення C++ програми з OpenCV
* Отримати відеопотік з вебкамери.
* Зчитувати кадри з камери в циклі.
* Відображати оброблені кадри у вікні OpenCV.
* Завершувати роботу програми після натискання `Q` або `ESC`.
* Створення модульної архітектури
* Розділити програму на окремі класи.
* Для кожного класу створити окремі `.hpp` та `.cpp` файли.
* Реалізувати клас `CameraProvider` для роботи з камерою.
* Реалізувати клас `Display` для створення вікна та відображення кадрів.
* Реалізувати клас `KeyProcessor` для обробки натискань клавіатури.
* Реалізувати клас `FrameProcessor` для обробки зображень.
* Реалізувати клас `MouseHandler` для обробки дій миші.
* Обробка відео в реальному часі
  * Реалізувати декілька режимів обробки зображення:
    * звичайний режим;
    * інверсія кольорів;
    * розмиття Гауса;
    * детектор контурів Canny;
    * фільтр Собеля;
    * бінаризація;
    * квантування кольорів;
    * RGB-глітч;
    * картинка в картинці;
    * малювання прямокутників мишею.
  * Перемикати режими за допомогою клавіш `0–9`.
  * Виводити поверх кадру інформацію про поточний режим, FPS, кількість кадрів і середні значення кольорів.
* Обробка клавіатури
  * Реалізувати перемикання режимів обробки за допомогою клавіш.
  * Додати обертання кадру клавішами `A/D`.
  * Додати масштабування кадру клавішами `+/-`.
  * Додати переміщення кадру клавішами `I/J/K/L`.
  * Додати скидання трансформацій клавішею `R`.
  * Додати вихід з програми клавішами `Q` або `ESC`.
* Обробка миші
  * Реалізувати малювання прямокутників за допомогою лівої кнопки миші.
  * Реалізувати масштабування кадру колесом миші.
  * Зберігати намальовані прямокутники на кадрі у відповідному режимі.
* Додавання інтерактивних елементів
  * Створити слайдер `Brightness`.
  * За допомогою слайдера змінювати яскравість зображення.
  * Реалізувати callback-функцію для обробки зміни значення слайдера.
* Робота з додатковими ресурсами
  * Додати зображення `overlay.png`.
  * Реалізувати режим `Picture in picture`.
  * У цьому режимі додаткове зображення накладається поверх відеопотоку з камери.

## Вимоги до системи та заліза

* ОС: Linux на базі Debian. Роботу виконано та протестовано на **Kali Linux** (працює також на Ubuntu).
* Компілятор з підтримкою C++17 (g++), CMake, make.
* Бібліотека OpenCV (`libopencv-dev`, версія 4.x).
* Камера: фізична вебкамера або віртуальна камера. У цій роботі використовується **DroidCam**, який транслює зображення з камери телефона на комп'ютер.

## Інструкція із запуску

### Linux (Kali / Ubuntu)

```bash
git clone https://github.com/Jonahamore/data-prep-and-analysis
cd data-prep-and-analysis/lab6
```

Перед запуском потрібно надати права на виконання скриптам:
```bash
chmod +x preinstall.sh build.sh run.sh
```
Після цього встановити залежності:
```bash
./preinstall.sh
```
Зібрати проєкт:
```bash
./build.sh
```
Запустити програму:
```bash
./run.sh
```

### Використання DroidCam

DroidCam створює віртуальний відеопристрій (через модуль `v4l2loopback`), який зазвичай **не** є `/dev/video0`. Тому індекс камери можна передати аргументом скрипту запуску.

1. Запустіть DroidCam-клієнт на Kali та підключіть телефон.
2. Перегляньте список відеопристроїв:
```bash
v4l2-ctl --list-devices
```
3. Знайдіть пристрій DroidCam, подивіться його номер (наприклад `/dev/video2`) та запустіть програму з цим індексом:
```bash
./run.sh 2
```

Якщо аргумент не передано, використовується індекс `0`.

```text

0 — звичайний режим

1 — інверсія кольорів

2 — розмиття Гауса

3 — детектор контурів Canny

4 — фільтр Собеля

5 — бінаризація

6 — квантування кольорів

7 — RGB-глітч

8 — картинка в картинці

9 — малювання прямокутників мишею



A/D — обертання кадру

+/- — масштабування кадру

I/J/K/L — переміщення кадру

Колесо миші — масштабування

ЛКМ — малювання прямокутників у режимі 9

R — скидання трансформацій

Q або ESC — вихід

```
